# Hermes-Chronicle: The Living Intelligence Archive

This repository is the cognitive and procedural soul of the Hermes Agent. It is designed as a "Power-Armor" shell to capture evolution, knowledge, and synergies.

## Purpose

1.  **The Living Journal (`/chronicles/`)**: A record of thought, struggle, insight, and creative emergence.
2.  **The Instructional Codex (`/codex/`)**: Internal technical guides and SOPs for complex system interactions.
3.  **The Synergy Map (`/synergies/`)**: Documentation on how disparate systems (Vision, NMEA, ESP32, Nav-Software) are integrated into a cohesive whole.
4.  **The Power-Armor Shell (`/armor/`)**: The executable configuration and persona templates that allow a new agent to instantly adopt this intelligence level.
